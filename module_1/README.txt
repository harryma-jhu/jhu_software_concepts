Project Title: Module 1 - Personal Website 

1. Project Overview
This is a Flask-based web serving as a professional portfolio. It features a home page, projects page, and contact info page. 

2. System Requirements
- Python: 3.10+ 
- Libraried listed under requirements.txt installed 

3. Set Up/Installs 
1) Extract the files into a local directory.
2) Create a virtual environment 
3) pip install -r requirements.txt

4. Run Application 
Execute 'python run.py' in terminal

5. Project Structure
- run.py: Main exectuable 
- backbone: Container for app 
    - static: Conatains css styling and images 
    - templates: Contains HTML files for each page and fixed.html (items that remain the same on each page)
    - routes.py: Blueprints 
- requirements.txt: List of libraries essential for app reconstruction
